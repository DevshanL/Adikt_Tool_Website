Project Scope

The ADIKT tool is developed to achieve the main research goal which is to, Increase the productivity and production of apparel industry. To achieve this goal, the main research component was divided into four different components which are to, Personality Prediction and Confidence Level Identifier using Natural Language Processing, Analyzing user trends and giving insights, Capturing apparel items, Identifying fashion items, and provide recommendations, Forecasting Marketing Suitability: Evaluating Products for Advertising Campaigns. All these four components are handled by trained models by using different algorithms. This research contains Natural language processing (NLP) and Image Processing (IP)

Research Gap
